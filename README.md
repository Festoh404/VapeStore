A modern, highly interactive front-end e-commerce website built for a premium shopping experience. This project demonstrates responsive design, advanced CSS animations, and state management using Vanilla JavaScript.

## ✨ Key Features

* **Interactive Slide-Out Cart:** A custom-built cart drawer that slides in seamlessly without interrupting the user's shopping experience.
* **Persistent State Management:** Utilizes browser `localStorage` to save cart data, ensuring items remain in the cart even after page refreshes.
* **Dynamic Cart Logic:** Includes real-time price calculations, quantity adjustments (+/-), and item removal functionalities.
* **Glassmorphism UI:** Features premium frosted-glass overlays and blur effects built purely with CSS (`backdrop-filter`) for product cards.
* **Custom CSS Animations:** Implements a "zero-gravity" floating image effect using `@keyframes` and smooth 3D box-shadow transitions.
* **Typewriter Text Effect:** A lightweight JavaScript interval function that dynamically types and erases promotional copy in the hero section.
* **Fully Responsive:** Mobile-first flexbox layout that scales perfectly across desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

* **HTML5:** Semantic markup for accessibility and SEO.
* **CSS3:** Custom variables, Flexbox, transitions, and keyframe animations.
* **Vanilla JavaScript (ES6+):** DOM manipulation, event listeners, array methods, and local storage (No jQuery or external libraries).
